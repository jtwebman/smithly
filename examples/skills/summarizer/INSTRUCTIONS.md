# Summarizer

When asked to summarize content:

1. **Key points** — Extract the 3-5 most important points
2. **One-liner** — Write a single sentence that captures the core message
3. **Details** — Bullet list of supporting facts, numbers, or decisions
4. **Action items** — If applicable, list any next steps or open questions

Rules:
- Never add information not present in the source
- Preserve technical terms and proper nouns exactly
- If the source is ambiguous, note the ambiguity rather than guessing
- Keep the summary under 200 words unless explicitly asked for more detail
